<h1 align="center">📦 Tableau: Amazon Sales Analysis Dashboard</h1>

<h2>📌 Project Overview</h2>
<p>
This project presents an interactive <b>Amazon Sales Analysis Dashboard</b> developed using
Tableau. The dashboard analyzes Amazon sales data to provide valuable insights into sales
performance, profit trends, product categories, regional sales, customer behavior, and order
distribution. It enables businesses to monitor key performance indicators (KPIs) and make
data-driven decisions through interactive visualizations.
</p>

<h2>🎯 Problem Statement</h2>
<p>
E-commerce businesses generate massive amounts of sales data every day, making it difficult
to manually identify trends and business opportunities. This project provides an interactive
Tableau dashboard that helps analyze sales performance, product profitability, customer
purchasing behavior, and regional performance, enabling better strategic planning and
business growth.
</p>
